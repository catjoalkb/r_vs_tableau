[`R_vs_Tableau.html`](https://catjoalkb.github.io/r_vs_tableau/R_vs_Tableau.html) 是最终文章内容。你也可以 [下载整个 Repo](https://github.com/catjoalkb/r_vs_tableau/archive/master.zip) 后双击打开 `R_vs_Tableau.html` 查看。
